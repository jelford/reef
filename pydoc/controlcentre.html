controlcentre Module¶

synopsis:Interfaces with the vazels control centre.

Not much in here. Most of the dealing (including stopping the control centre) is done by the command client in commandclient.

controlcentre.startVazels()¶

Start the vazels control centre.

This should also set up the tear down when the control centre closes.

Returns:True if the control centre is still running when we exit the function. (i.e. nothing has gone wrong yet.)
Return type:bool
controlcentre.runningState()¶

Check what state the vazels process is in.

Returns:
  • True if the control centre is running.
  • False if the control centre is not running (or we don’t know its running...).
  • None if we are in an intermediate state.
Return type:bool
Raises :IOError if the vazels output could not be read.
controlcentre.getVazelsPath()¶

Grab the path to the vazels installation directory.

Returns:Absolute path to the vazels install directory.
Return type:str
controlcentre.getExperimentPath()¶

Grab the experiment path.

Returns:Absolute path to the vazels experiment for this project.
Return type:str
controlcentre.getControlCentreOutPath()¶

Grab the control centre output file.

Returns:Absolute path to the ouput file.
Return type:str
