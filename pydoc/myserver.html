myserver Module¶

synopsis:Quick and easy implementation of a restlite-based server.

This was implemented so that the server acts as a class but should only be created with getServer() so remains singleton.

class myserver.ReefServer¶

Represents the server.

This should perform setup and running operations with minimal input from elsewhere.

setup()¶

Perform any setup needed to bend the server to our whims.

This really only grabs the routing table from the handlers module.

start()¶

Build the actual server and set it running.

This will use the setup performed by any of the other class methods, it also takes into account any config values set.

myserver.getServer()¶

Grab the singleton server instance.

Returns:A server instance
Return type:ReefServer
