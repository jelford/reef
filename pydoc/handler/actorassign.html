handler.actorassign Module¶

synopsis:Handler to allow assignment of actors to workloads.
class handler.actorassign.ActorAssignHandler¶

Bases: handler.dochandler.DocHandler

Documented handler for assignment of actors to workloads.

POST(request, entity)¶

(Un)assign actors (from/)to workloads.

The parameters listed are part of the url encoded entity.

Parameters:
  • do – What to do, either "add" or "rem" to add or remove respectively
  • workload – The name of the workload to affect.
  • actor – The name of the actor to add or remove.
Raises :

restlite.Status 400 if a parameter is given multiple times.

restlite.Status 400 if a parameter is missing.

restlite.Status 404 if the actor or workload does not exist.

restlite.Status 400 if the action (do) is invalid.

restlite.Status 409 if the action cannot be completed do to the state of currently attached workloads.
Returns:

The JSON representation of the workload.
addActor(workload, actor)¶

Add an actor to the given workload.

Parameters:
  • workload (dict) – The workload (from config) to add to.
  • actor (str) – The name of the actor to add.
Raises :

restlite.Status 409 if the actor is already attached.
remActor(workload, actor)¶

Remove an actor from the given workload.

Parameters:
  • workload (dict) – The workload (from config) to remove from.
  • actor (str) – The name of the actor to remove.
Raises :

restlite.Status 409 if the actor to remove is not attached.
