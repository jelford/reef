handler.auth_page Module¶

synopsis:Handles basic authenticated pages requests.
handler.auth_page.MIMETYPES¶

A dictionary matching file extensions to mimetypes.

class handler.auth_page.AuthPageHandler¶

Bases: handler.dochandler.DocHandler

Handles normal page requests and requires authentication from the requester.

Authentication is dealt with by the authentication module.

GET(request)¶

Get a particular page.

The handler will take anything after the base address and look for it in the pages folder. This folder is stored in the config module at:

config.getSettings("server")["pagedir"]

If a file is not found the handler will assume the address is a directory and try to serve the file index.html from it.

Returns:

The requested file.
Raises :

restlite.Status 400 if the path contains ”..”

restlite.Status 404 if the file is not found.
