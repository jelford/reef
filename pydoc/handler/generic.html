handler.generic Module¶

synopsis:Replaces the restlite usage message.
class handler.generic.GenericPageHandler¶

Bases: handler.dochandler.DocHandler

Used to replace restlite‘s usage message when a request is not handled.

Instead, if inserted to handle pages at “/”, this will return 404 for any GET request.

GET(request)¶

Return a 404 error.

Raise :restlite.Status 404...always!
