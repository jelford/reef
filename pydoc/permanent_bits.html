permanent_bits Module¶

synopsis:Holds constants to be used throughout the application.
permanent_bits.appname¶

The short name of the application.

permanent_bits.appname_file¶

The representation of the application to be used in file names.

permanent_bits.appname_long¶

The long version of the application title.

permanent_bits.app_desc¶

The description for the application, used as a small blurb.

permanent_bits.app_user¶

A user name to be used by default by the application.
