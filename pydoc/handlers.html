handlers Module¶

synopsis:Supplies reef server routing table.

This is used by the server when setting up. All it does it provide a nice place for us to add new handlers without delving through a pile of code to get there.

handlers.getRouting()¶

Get the routing table for the server.

This will be in the format used in restlite.router().

Returns:A routing table to be used by the reef backend.
Return type:list of tuple as in restlite.router().
