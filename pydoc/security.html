security Module¶

synopsis:Deals with security for the server.

This mainly involves setting up and tearing down ssh authentication used by vazels.

security.makeKey()¶

Generate a new ssh key to be used by vazels.

This will be very insecure as Vazels requires a passwordless key. If you wish to inspect the generated product, it will be saved to the path returned by keyPath().

security.authoriseKey()¶

Add our key to the ssh authorized_hosts file.

This will first back up the current file so we can’t damage anything too badly. This means we should be safe if we forgot to create the key as we’ll just receive an error message and the backup will be restored.

security.backup()¶

Back up the ssh authorized_keys file if it is not already backed up.

You can then fiddle about with it and restore the backup when you’re done using restore().

security.restore()¶

Restore the backed up version of the ssh authorized_hosts file if a backup has been created by backup().

security.authKeysFile()¶

Get the location of ssh‘s authorized_keys file.

security.backupFile()¶

Get the location to store or retrieve our backup file.

This is used by backup() and restore().

Returns:Absolute path to our authorized_hosts backup file.
Return type:str
security.keyPath()¶

Get the location of our generated key file.

Returns:Absolute path to our generated key file.
Return type:str
security.securityDir()¶

Get the location of our security directory.

This should be somewhere inside our project directory as described here.

Returns:Absolute path to the security directory.
Return type:str
security.ensureSecurityDir()¶

Make sure the security directory exists.
