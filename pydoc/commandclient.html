commandclient.issueControlCentreCommand(command, args=[], extraargs=[], afterwards=None)¶

Send a command to the command client and optionally run a function on finish.

Parameters:
  • command (str) – The command to run.
  • args (list(str)) – List of arguments to the command. (optional)
  • extraargs (dict) – Arguments to the script, i.e. those beginning with “–”. (optional)
  • afterwards (func, takes the output file as a parameter.) – The function to run afterwards (optional).
Returns:

True if the control centre is running, False otherwise.
Return type:

bool
